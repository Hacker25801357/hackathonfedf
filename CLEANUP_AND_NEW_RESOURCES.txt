HOW TO ADD MORE RESOURCES:
================================================================

1. CREATE PDF FILE:
   - Prepare your educational resource as a PDF file

2. UPLOAD FILE:
   - POST to /api/resources/upload with file in form data
   - Include admin authorization header
   - Receive fileUrl, fileName, fileSize in response

3. CREATE RESOURCE:
   - POST to /api/resources with resource metadata
   - Include fileUrl, fileName, fileSize from upload response
   - Add title, description, subject, type, tags

4. VERIFY:
   - Check that resource appears in /api/resources
   - Verify file is accessible via browser

================================================================
COMMANDS USED:
================================================================

DELETE DUMMY RESOURCES:
curl -X DELETE -H "Authorization: Bearer [TOKEN]" http://localhost:5000/api/resources/[RESOURCE_ID]

UPLOAD FILE:
curl -X POST -H "Authorization: Bearer [TOKEN]" -F "file=@filename.pdf" http://localhost:5000/api/resources/upload

CREATE RESOURCE:
curl -X POST -H "Content-Type: application/json" -H "Authorization: Bearer [TOKEN]" -d "{\"title\":\"...\",\"type\":\"...\",\"description\":\"...\",\"subject\":\"...\",\"fileUrl\":\"...\",\"fileName\":\"...\",\"fileSize\":...}" http://localhost:5000/api/resources
